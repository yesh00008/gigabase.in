To run this program, download and install the latest GNU Make using these
instructions:

* [Windows][7]
* For [Linux][8], see "How to Download" section
* [Mac][9]

Download a copy of [Hello World in GNU Make][10], and run this command:

```
make -sf hello-world.mk
```
